# Retail-Backend

Back-end của dự án Unified Retail — khởi tạo bằng NestJS + TypeScript.

## Yêu cầu

- Node.js
- MySQL

## Cài đặt

1. Cài dependencies:

```
npm install
```

2. Tạo file cấu hình môi trường:

```
cp .env.example .env
```

Sau đó chỉnh file .env theo ví dụ

3. Chạy ở chế độ phát triển:

```
npm run start:dev
```

4. Build & chạy production

```
npm run build
npm run start:prod
```
